Ein Gutenberg-Block für die Suchleiste mit fuzzy Suche in FAQs.

== Description ==

Der FAQ Search Block bietet eine Suchleiste für client-side fuzzy Suche in FAQs. Die Suche filtert die FAQs in Echtzeit ohne Seitenneuladen.

Ideal für:

* **Dokumentationsseiten** - Schnelle Navigation in großen FAQ-Listen
* **Support-Bereiche** - Sofortige Ergebnisse für Benutzerfragen
* **Schulungsportale** - Interaktive Hilfesuche

== Features ==

* **Fuzzy Suche** - Toleriert Tippfehler und Variationen
* **Client-Side** - Kein Seitenneuladen, sofortige Filterung
* **Deutsche Benutzeroberfläche** - Vollständig lokalisiert
* **Integration mit FAQ Display** - Filtert verbundene Akkordeons
* **Responsive Design** - Funktioniert auf allen Geräten
* **Barrierefreiheit** - Unterstützt Screenreader und Tastaturnavigation

== Installation ==

1. Den Block-Ordner in das WordPress-Plugin-Verzeichnis hochladen
2. Block über WordPress-Admin aktivieren
3. Block im Gutenberg-Editor verwenden

== Frequently Asked Questions ==

= Wie verwende ich den Block? =

1. Neuen Block hinzufügen und "FAQ Search" auswählen
2. Target-Display-ID eingeben (ID des FAQ Display Blocks)
3. Block über dem FAQ Display platzieren

= Wie funktioniert die Fuzzy-Suche? =

Die Suche verwendet Fuse.js für intelligente Übereinstimmungen in Titeln und Inhalten.

= Ist der Block barrierefrei? =

Ja, der Block unterstützt Tastaturnavigation und Screenreader.
