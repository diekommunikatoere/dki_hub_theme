Ein Gutenberg-Block zur Anzeige von FAQs in nested Akkordeons mit Kategorien.

== Description ==

Der FAQ Display Block zeigt FAQs strukturiert als nested Akkordeons an, mit Abschnitten (Kategorien) als äußeren Akkordeons und individuellen FAQ-Items als inneren. Ideal für:

* **Dokumentationsseiten** - Organisierte Wissensdatenbank
* **Support-Bereiche** - Schnelle Antworten auf häufige Fragen
* **Schulungsportale** - Kategorisierte Hilfestellungen

== Features ==

* **Nested Akkordeons** - Abschnitte enthalten FAQ-Items
* **Automatische Query** - Lädt alle FAQs aus dem 'faq' CPT
* **Deutsche Benutzeroberfläche** - Vollständig lokalisiert
* **Anzeige-Only** - Keine Inline-Bearbeitung, reine Darstellung
* **Responsive Design** - Funktioniert auf allen Geräten
* **Barrierefreiheit** - Unterstützt Screenreader und Tastaturnavigation

== Installation ==

1. Den Block-Ordner in das WordPress-Plugin-Verzeichnis hochladen
2. Block über WordPress-Admin aktivieren
3. Block im Gutenberg-Editor verwenden

== Frequently Asked Questions ==

= Wie verwende ich den Block? =

1. Neuen Block hinzufügen und "FAQ Display" auswählen
2. Block einfügen auf der Seite
3. FAQs werden automatisch aus dem CPT geladen

= Welche Einstellungen gibt es? =

Der Block ist display-only mit minimalen Attributen für Stil-Anpassungen.

= Ist der Block barrierefrei? =

Ja, der Block unterstützt Tastaturnavigation und Screenreader.
