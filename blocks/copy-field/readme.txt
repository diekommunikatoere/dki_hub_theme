Ein Gutenberg-Block zum Kopieren von formatiertem Text und Code-Snippets.

== Description ==

Der Copy Field Block ermöglicht es Benutzern, Inhalte einfach in die Zwischenablage zu kopieren. Ideal für:

* **Textvorlagen** - Formatierte E-Mail-Antworten, Dokumentationsauszüge
* **Code-Snippets** - Befehle, Konfigurationen, API-Beispiele
* **Schulungsunterlagen** - Wiederverwendbare Inhaltsblöcke

== Features ==

* **Rich Text Editor** (Standard) - Für formatierte Textvorlagen
* **Code Block Editor** - Für technische Inhalte mit Syntax-Highlighting
* **Deutsche Benutzeroberfläche** - Vollständig lokalisiert
* **Nur-Lese-Modus** - Für Anzeigefelder ohne Bearbeitung
* **Responsive Design** - Funktioniert auf allen Geräten
* **Barrierefreiheit** - Unterstützt Screenreader und Tastaturnavigation

== Installation ==

1. Den Block-Ordner in das WordPress-Plugin-Verzeichnis hochladen
2. Block über WordPress-Admin aktivieren
3. Block im Gutenberg-Editor verwenden

== Frequently Asked Questions ==

= Wie verwende ich den Block? =

1. Neuen Block hinzufügen und "Copy Field" auswählen
2. Eingabetyp wählen (Rich Text oder Code)
3. Inhalt eingeben
4. Benutzer können den "Kopieren"-Button verwenden

= Welche Code-Sprachen werden unterstützt? =

Bash/Shell, JavaScript, PHP, CSS, HTML, JSON, SQL, Python

= Ist der Block barrierefrei? =

Ja, der Block unterstützt Tastaturnavigation und Screenreader.
